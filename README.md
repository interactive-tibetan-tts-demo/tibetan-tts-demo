# Interactive Tibetan Text-to-Speech Demo

## Online Demo

[Open the Tibetan TTS Demo](https://interactive-tibetan-tts-demo.github.io/tibetan-tts-demo/)

## How to Use

1. Open the online demo.
2. Enter any Tibetan text in the input box.
3. Click **Synthesize audio**.
4. Wait for the synthesis to complete.
5. Play the generated audio or download the WAV file.

## Usage Limit

Each IP address can successfully synthesize audio up to 10 times.

## Browser Requirements

Use a recent version of Chrome, Edge, Firefox, or Safari. An internet connection is required.
